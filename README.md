# RLT.HACK 2026

Система для автоматизированного подбора закупок, которые соответствуют товарной матрице поставщика. Разработана в рамках хакатона [RLT.HACK](https://uni.roseltorg.ru/intensive/intellektualnyj-servis-poiska-zakupok-associirovannyh-s-nomenklaturnoj-matricej-postavshika) с целью улучшить поиск релевантных закупок для селлеров маркетплейсов или интернет-магазинов, основываясь на их номенклатуре.

---
### Реализованный функционал:
- обработка CSV с товарами поставщика и создание текстов для поиска
- генерация плотных векторных представлений товаров с помощью модели BGE-M3
- применение RRF для слияния результатов поиска из двух индексов
- ранжирование лотов по общей оценке соответствия
- генерация кратких текстовых описаний лотов с использованием LLM
- группировка результатов по идентификатору лота для оптимизированного поиска

### Технологии и инструменты:
- deepv/USER-bge-m3 (dense, coIBERT)
- FAISS
- Qwen3-4b
- BAAI/bge-reranker-v2-m3
- Python
- FastAPI
- JavaScript
- Vue

1. Backend:
